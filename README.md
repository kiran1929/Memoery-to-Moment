## 📖 About The Project

Traditional event planning is highly fragmented, bloated, and forces users to deal with expensive agency middlemen. 

**The Big Reveal** is a D2C platform that completely re-engineers how people gift experiences. We connect customers directly with independent "Memory Makers" (stealth photographers, videographers) and local artisans to seamlessly execute and capture surprise events. 

### 🎯 The Problem We Solve
* **The Middleman Markup:** Event agencies take 30-40% cuts. We eliminate them.
* **The Logistics Nightmare:** Coordinating a surprise without ruining it is incredibly difficult for a layman.
* **The Missed Moment:** People spend thousands on gifts, but rely on blurry smartphone photos to capture the reaction.

## ✨ Core Features

### 1. 🤫 The Surprise Planner (Discovery & Setup)
* **Vibe-Based Discovery:** Browse D2C surprise packages based on emotion (e.g., "Midnight Ambush", "Luxury Rooftop Proposal").
* **Stealth Dashboard:** A disguised UI that allows users to manage bookings and chat with creators without the person being surprised seeing a notification.

### 2. 📸 The Memory Maker (Creator Network)
* **Instant Social Bundles:** Book stealth creators for rapid 24-hour turnaround Reels and raw POV photos.
* **Legacy Documentary Bundles:** Book high-end crews for multi-day cinematic coverage.
* **Direct Storefronts:** Creators keep 100% of their listed price with transparent up-front booking. 

### 3. ⚙️ Event Fulfillment (The D2C Engine)
* **Smart Micro-Contracts:** Friction-free digital agreements between the buyer and the creator generated at checkout.
* **The Memory Vault:** A secure, built-in digital gallery where creators upload media directly to the customer post-event.

## 💻 Tech Stack

This project was built using modern web technologies to ensure a scalable and smooth user experience:

* **Frontend:** React.js, Tailwind CSS (for rapid, responsive UI styling)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (NoSQL for flexible package and user profile structures)
* **Design/Prototyping:** Figma

---

## 🏗 System Architecture

**The Big Reveal** operates on a decoupled client-server architecture:
1. **Client Interface:** A React single-page application handling the "Stealth Dashboard" and marketplace browsing.
2. **API Gateway:** An Express.js backend processing authentication, booking logic, and direct messaging between customers and creators.
3. **Data Layer:** MongoDB stores three primary collections: `Users` (Planners & Creators), `Packages` (Services offered), and `Bookings` (Active event connections).
